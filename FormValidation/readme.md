# Form Validator

A live form validator using HTML, CSS, and JavaScript with:

- Real-time validation for Name, Phone, Email, Password, and Message
- Password strength checker (Weak / Medium / Strong)
- Password visibility toggle
- Responsive and clean UI
- Error messages and validation hints

### Features

- Full Name validation (must include first and last name)
- Phone number validation (10-digit numeric)
- Email validation (regex)
- Password strength meter
- Password show/hide toggle
- Message minimum character requirement
- Submit button only works if all fields are valid
