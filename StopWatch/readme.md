# Stopwatch App

A simple and interactive **Stopwatch** built using **HTML, CSS, and JavaScript**. This app allows you to **start, stop, and reset** a stopwatch timer. Perfect for learning basic DOM manipulation and timer logic in JavaScript.

## Features

- Displays hours, minutes, and seconds.
- **Start** button to begin counting.
- **Stop** button to pause the timer.
- **Reset** button to reset the timer to `00:00:00`.
- Stylish, minimal UI with centered layout.

## How to Use

1. Open `index.html` in a web browser.
2. Click **Start** to begin the stopwatch.
3. Click **Stop** to pause the stopwatch.
4. Click **Reset** to clear the timer.

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM manipulation, `setInterval`, `clearInterval`)
