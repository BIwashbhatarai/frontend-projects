# Horizontal Scroll Image Gallery

A sleek **horizontal image gallery** built with **HTML, CSS, and JavaScript**. Users can scroll through images using the **mouse wheel** or navigation **buttons**.

---

## Features

- **Horizontal scrolling** with mouse wheel
- **Next/Back buttons** for easy navigation
- **Smooth scrolling** animation for button clicks
- **Responsive gallery layout**
- **Hover effect**: images scale up and remove grayscale on hover
- Custom scrollbar hidden for a clean look
