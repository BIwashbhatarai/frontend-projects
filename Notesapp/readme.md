# Notes App

A simple, colorful, and interactive **Notes App** built with HTML, CSS, and JavaScript. This app allows users to create, edit, and delete notes with persistence using `localStorage`.

---

## Features

- **Create Notes**: Click the **Create Note** button to add a new editable note.
- **Edit Notes**: Notes are content-editable. You can type directly inside them.
- **Delete Notes**: Click the delete icon on any note to remove it.
- **Persistent Storage**: Notes are saved in `localStorage`, so they remain even after refreshing the page.
- **Responsive Grid Layout**: Notes are displayed in a neat grid layout.
- **Keyboard Support**: Pressing **Enter** adds a new line inside the note instead of creating a new note.
