# Todo App

A simple **Todo List** application built using **HTML, CSS, and JavaScript**. Users can **add, mark as completed, and delete tasks**. The tasks are saved in **localStorage** so they persist across page reloads.

## Features

- Add new tasks to the list.
- Mark tasks as completed by clicking on them.
- Delete tasks using the delete (×) button.
- Tasks are saved in **localStorage** and persist after refreshing the page.
- User-friendly interface with a clean, responsive design.

## How to Use

1. Open `index.html` in your browser.
2. Type your task in the input box and click **Add**.
3. Click on a task to mark it as completed.
4. Click on the **×** icon to delete a task.
5. Your tasks are automatically saved in the browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM manipulation & localStorage)
